# 算出プロパティとウォッチャー

* Vueインスタンス内のcomputedオプションで記述される内容が、算出プロパティ
* リアクティブな依存関係（dataに記述された内容）が更新されたときのみ、算出プロパティが更新される。
* watchを使うと、選択したリアクティブな依存関係が変更されたときに起こる挙動を記述できる。
